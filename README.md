# Módulo 13 – NLP Aplicaciones Clásicas

Este repositorio contiene los notebooks desarrollados para el **Módulo 13 de Data Science (NLP)**, con ejemplos prácticos de aplicaciones clásicas de Procesamiento de Lenguaje Natural.

## 📂 Contenido

Cada notebook corresponde a un escenario de aplicación:

1. **Escenario 1 – Opinión de clientes (Análisis de Sentimiento)**  
   Uso de `TextBlob` para detectar polaridad en comentarios de clientes.

2. **Escenario 2 – Organización de noticias (Clasificación de texto)**  
   Clasificación automática de artículos en categorías simples (economía, deportes, otros).

3. **Escenario 3 – Revisión rápida de contenido (Resumen automático)**  
   Generación de resúmenes extractivos con `sumy`.

4. **Escenario 4 – Automatización de soporte (Chatbots)**  
   Chatbot basado en reglas para responder preguntas frecuentes.

5. **Escenario 5 – Detección de spam en correos electrónicos**  
   Ejemplo de filtrado de mensajes usando técnicas de NLP.
